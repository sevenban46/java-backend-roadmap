# Ejercicios Java

Estas carpetas organizan tu practica de las semanas 1 a 8.

## Proyecto Maven base
Este directorio ya incluye un proyecto Maven funcional para que practiques sin perder tiempo en configuracion.

## Estructura sugerida
- `src/main/java/com/alexc/ejercicios/week01` a `week08`: usa cada paquete para los ejercicios de la semana correspondiente.
- `src/test/java`: escribe pruebas sencillas cuando llegues a semana 7.
- `pom.xml`: base con Java 21 y JUnit 5.

## Comandos utiles
- `mvn compile`
- `mvn test`
- `mvn exec:java` no esta configurado todavia a proposito; primero practica lo esencial.

## Regla de oro
No copies soluciones. Lee el enunciado, disena una estrategia en papel, implementa, prueba y luego explica tu codigo con tus palabras.

## Como trabajar cada semana
1. Leer la guia de la semana.
2. Resolver primero los ejercicios obligatorios.
3. Si terminas, atacar los retos extra.
4. Subir tus soluciones al repositorio de ejercicios con commits pequenos.
5. Registrar dudas y errores en el diario.
