# Semana 8 - SQL Basico y Relacional

## Objetivo
Entender como guardar y consultar datos relacionales para preparar el salto a Spring Boot.

## Donde empezar
Trabaja dentro de `ejercicios-java/semana-08-sql/sql`.

Archivos creados para arrancar:
1. `01-create-tables.sql`
2. `02-insert-sample-data.sql`
3. `03-basic-queries.sql`
4. `04-joins.sql`

Orden recomendado para empezar:
1. `01-create-tables.sql`
2. `02-insert-sample-data.sql`
3. `03-basic-queries.sql`
4. `04-joins.sql`

## Tareas obligatorias
1. Crear una base de datos simple para usuarios y tareas, o productos y ventas.
2. Escribir sentencias `CREATE TABLE`.
3. Insertar datos de prueba.
4. Hacer consultas con `SELECT`.
5. Actualizar y eliminar datos.
6. Hacer al menos 2 consultas con `JOIN`.

## Evidencia minima
- Script SQL versionado.
- Consultas comentadas o explicadas.
- Comprension de claves primarias y foraneas.
