# Semana 4 - Herencia, Interfaces y Composicion

## Objetivo
Modelar problemas con relaciones entre clases sin abusar de la herencia.

## Donde empezar en codigo
Trabaja dentro de `src/main/java/com/alexc/ejercicios/week04`.

Clases creadas para arrancar:
1. `Pagable`
2. `EmpleadoBase`
3. `EmpleadoTiempoCompleto`
4. `EmpleadoPorHora`
5. `Libro`
6. `Biblioteca`
7. `GestorBibliotecaConsola`

Orden recomendado para empezar:
1. `Pagable`
2. `EmpleadoBase`
3. `EmpleadoTiempoCompleto`
4. `EmpleadoPorHora`
5. `Libro`
6. `Biblioteca`
7. `GestorBibliotecaConsola`

## Ejercicios obligatorios
1. Jerarquia de `Empleado` con subtipos.
2. Interface `Pagable` o `Imprimible` aplicada a varias clases.
3. Ejemplo de composicion con `Biblioteca` y `Libro`.
4. Uso simple de polimorfismo en una coleccion de objetos.

## Proyecto de semana
Crear un `Sistema de Biblioteca` o `Gestor de Tareas` en consola.

## Evidencia minima
- Al menos una interfaz.
- Al menos un ejemplo de herencia justificada.
- Un ejemplo claro de composicion.
