# Semana 1 - Fundamentos

## Objetivo
Dominar variables, tipos, operadores, condicionales y entrada/salida.

## Donde empezar en codigo
Trabaja dentro de `src/main/java/com/alexc/ejercicios/week01`.

Clases creadas para arrancar:
1. `CalculadoraBasica`
2. `ConversorTemperatura`
3. `ValidadorEdad`
4. `MayorDeDosNumeros`
5. `MenuBasico`
6. `PromedioNotas`
7. `NumeroParImpar`
8. `ConversorMoneda`
9. `CalculadoraDescuento`
10. `NumeroPositivoNegativoOCero`

Orden recomendado para hoy:
1. `CalculadoraBasica`
2. `MayorDeDosNumeros`
3. `NumeroParImpar`

## Ejercicios obligatorios
1. Calculadora basica con suma, resta, multiplicacion y division.
2. Conversor de Celsius a Fahrenheit.
3. Validador de mayoria de edad.
4. Programa que compare dos numeros y diga cual es mayor.
5. Menu simple con 3 opciones.
6. Calculadora de promedio de tres notas.
7. Verificador de numero par o impar.
8. Conversor de moneda usando una tasa fija.
9. Calculadora de descuento.
10. Verificador de si un numero es positivo, negativo o cero.

## Retos extra
- Simular un cajero con saldo inicial.
- Validar entrada incorrecta sin cerrar el programa.

## Evidencia minima
- 10 ejercicios terminados.
- Nombres de clases claros.
- Un commit por grupo pequeno de ejercicios.
