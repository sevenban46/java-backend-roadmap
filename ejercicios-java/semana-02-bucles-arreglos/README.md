# Semana 2 - Bucles, Metodos y Arreglos

## Objetivo
Practicar repeticion, modularizacion y manejo basico de datos en memoria.

## Donde empezar en codigo
Trabaja dentro de `src/main/java/com/alexc/ejercicios/week02`.

Clases creadas para arrancar:
1. `TablaMultiplicar`
2. `SumaPrimerosNNumeros`
3. `ContadorVocales`
4. `MayorEnArreglo`
5. `PromedioArreglo`
6. `ContadorNumerosPares`
7. `InvertirArreglo`
8. `BuscarElemento`
9. `FactorialConMetodo`
10. `MatrizTresPorTres`

Orden recomendado para empezar:
1. `TablaMultiplicar`
2. `SumaPrimerosNNumeros`
3. `ContadorVocales`
4. `FactorialConMetodo`

## Ejercicios obligatorios
1. Tabla de multiplicar de un numero.
2. Suma de los primeros N numeros.
3. Contador de vocales en una palabra.
4. Buscar el numero mayor en un arreglo.
5. Calcular promedio de un arreglo.
6. Contar numeros pares en una lista.
7. Invertir un arreglo.
8. Buscar si un elemento existe.
9. Crear un metodo para calcular factorial.
10. Imprimir una matriz 3x3.

## Retos extra
- Ordenar un arreglo sin usar utilidades avanzadas.
- Crear un menu repetitivo que permita varias operaciones.

## Evidencia minima
- 10 ejercicios terminados.
- Al menos 3 ejercicios con metodos separados.
- Explicacion escrita de `for`, `while` y arrays.
