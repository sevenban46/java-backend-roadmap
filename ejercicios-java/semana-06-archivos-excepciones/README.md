Volver tu codigo mas robusto y persistir informacion sin base de datos todavia.

## Ejercicios obligatorios
1. Manejo de entrada invalida con `try/catch`.
2. Guardar datos simples en archivo de texto.
3. Leer datos desde archivo y cargarlos en memoria.
4. Crear una excepcion personalizada sencilla.
5. Separar capas basicas: presentacion, logica y persistencia.

## Proyecto de semana
Aplicacion de consola con datos persistidos en archivos.

## Evidencia minima
- Manejo claro de errores.
- Lectura y escritura funcional.
- Separacion simple por responsabilidades.
