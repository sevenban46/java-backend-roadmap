# Semana 7 - Git, Maven y JUnit

## Objetivo
Trabajar de una manera parecida a un entorno real de desarrollo.

## Donde empezar en codigo
Trabaja en estas rutas:
- `src/main/java/com/alexc/ejercicios/week07`
- `src/test/java/com/alexc/ejercicios/week07`
- `ejercicios-java/semana-07-git-maven-junit/comandos-git-sugeridos.md`

Archivos creados para arrancar:
1. `CalculadoraUtil`
2. `CalculadoraUtilTest`
3. `comandos-git-sugeridos.md`

Orden recomendado para empezar:
1. Revisar `comandos-git-sugeridos.md`
2. Ejecutar `mvn test`
3. Leer `CalculadoraUtil`
4. Leer `CalculadoraUtilTest`
5. Crear tus propias pruebas y un commit pequeno

## Tareas obligatorias
1. Inicializar o limpiar tu repositorio de ejercicios.
2. Crear `.gitignore` adecuado para Java.
3. Convertir un proyecto previo a Maven.
4. Crear al menos 3 pruebas unitarias basicas con JUnit.
5. Escribir un README corto del proyecto elegido.

## Evidencia minima
- Commits pequenos.
- `pom.xml` funcionando.
- Pruebas ejecutandose sin errores.
