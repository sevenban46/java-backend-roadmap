1. Agenda de contactos con `ArrayList`.
2. Inventario usando `HashMap`.
3. Ranking de notas con ordenamiento simple.
4. Eliminar duplicados usando `Set`.
5. Buscar y filtrar elementos de una lista.

## Proyecto de semana
Refactorizar uno de tus proyectos anteriores para usar colecciones en lugar de arreglos cuando tenga sentido.

- Explicar cuando usar `List`, `Set` y `Map`.
- Aplicar recorrido y filtrado con claridad.
