# Semana 3 - Programacion Orientada a Objetos

## Objetivo
Pensar en terminos de clases, objetos y responsabilidades.

## Donde empezar en codigo
Trabaja dentro de `src/main/java/com/alexc/ejercicios/week03`.

Clases creadas para arrancar:
1. `Producto`
2. `CuentaBancaria`
3. `Estudiante`
4. `Empleado`
5. `SistemaGestionConsola`

Orden recomendado para empezar:
1. `Producto`
2. `CuentaBancaria`
3. `Estudiante`
4. `Empleado`
5. `SistemaGestionConsola`

## Modelos obligatorios
1. `Producto` con nombre, precio y stock.
2. `CuentaBancaria` con deposito y retiro.
3. `Estudiante` con nombre, edad y promedio.
4. `Empleado` con salario y metodo de aumento.

## Mini sistema obligatorio
Construir un programa de consola que use al menos 3 clases y muestre colaboracion entre objetos.

## Retos extra
- Agregar validaciones simples.
- Separar clase principal de clases de dominio.

## Evidencia minima
- Uso de constructores.
- Atributos encapsulados.
- Metodos con responsabilidad clara.
