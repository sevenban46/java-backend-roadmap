# Inventory System

## Rol en el portafolio
Este proyecto debe mostrar que ya puedes manejar un dominio con mas reglas que un CRUD simple.

## Alcance minimo
- Productos.
- Stock actual.
- Movimientos de entrada y salida.
- Busqueda o filtro por nombre, categoria o disponibilidad.
- Persistencia en base de datos.
- Pruebas basicas.

## Meta de aprendizaje
Practicar modelado de entidades relacionadas y reglas de negocio concretas.
