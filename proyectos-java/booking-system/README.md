# Booking System

## Rol en el portafolio
Este proyecto debe sentirse mas cercano a una aplicacion real y cerrar tu portafolio con mejor modelado, relaciones y reglas.

## Dominios posibles
- Reservas de salas.
- Turnos medicos.
- Prestamo de libros.

Elige uno y mantenlo consistente.

## Meta de aprendizaje
Demostrar que puedes modelar un dominio mas rico, manejar restricciones y explicar decisiones tecnicas.
